artefakt
Image Manager · artefakt.foundation

Place your photographs.

Drag an image file onto any slot below, or click a slot to browse. When you're done, click Save All to Repo — it writes directly into your ARTEFAKT-FOUNDATION/images folder. Then push to GitHub as usual.

How it works

Each slot corresponds to an image filename the site already expects. Drop the right photograph onto the right slot. You can replace an image at any time by dropping a new one. Nothing is saved until you click the Save button.

Save & deploy.

Click Save All to write your images into the repo's images folder. Then push to GitHub and Vercel deploys automatically. You can also drag a whole folder of images onto any slot — it'll pick the first image in the folder.

$ cd /Users/jonlowenstein/ARTEFAKT-FOUNDATION
$ git add images/
$ git commit -m "Update photographs"
$ git push
